Rules for Mega Book Project

  • I will use HTML, CSS (Tailwind) and javascript for this project only. No database etc will be used. All the files must be in HTML or md format.
  • This project should always be kept in a shape which can just be dragged and dropped into any wibsite as static pages.
  • Any javascript tool can be added to this project e.g svelte.
  • This will form part of my future websites projects.
  • Try to keep each article as ready to publish as possible
  • Use git to statically publish this project.
